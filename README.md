# Wallet Connect Demo

A React-based demo application showcasing wallet connectivity using WalletConnect protocol.

## Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- WalletConnect

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`

## Features

- Connect to various Web3 wallets
- Send tokens
- Modern UI with cyberpunk theme
